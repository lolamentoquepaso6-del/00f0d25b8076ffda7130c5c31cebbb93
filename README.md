
STREMIO STATIC ADDON - Canales Deportivos (GitHub Pages structure)
-----------------------------------------------------------------

Estructura exacta para subir a GitHub y usar con Pages (branch main, root):
- manifest.json
- catalog/channel/sports.json
- meta/channel/<id>.json
- stream/channel/<id>.json

Subir archivos sueltos (no ZIP). Activar Pages: Settings -> Pages -> Source: main branch -> /(root).

Luego usar la URL:
https://<TU_USUARIO>.github.io/<TU_REPO>/manifest.json
como add-on remoto en Stremio.

Nota: mantuve exactamente las URLs y los íconos crudos que acordamos.
